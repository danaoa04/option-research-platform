# Replay Guide

Replay helps you inspect deterministic event history and compare research
branches.

## Covered capabilities

- replay sessions;
- event timelines;
- snapshots;
- decisions;
- branch creation;
- branch comparison;
- annotations and notes;
- reproducibility checksums.

## Recommended use

Use Replay when you want to explain:

- why a risk breach happened;
- how an alternative branch diverged;
- which event first changed the result;
- how a decision connects to the original checksum trail.
