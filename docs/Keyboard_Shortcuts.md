# Keyboard Shortcuts

Documented shortcuts are limited to stable, implemented behavior.

## Current shortcuts

- `⌘K` / `Ctrl+K`: open the command launcher.
- `Esc`: close the command launcher or return workspace tab views to their
  default overview in supported pages.
- `⌘S` / `Ctrl+S`: store the current fixture tab state locally in supported
  strategy, research, risk, and volatility workspaces.

No other shortcuts are documented here because they are not yet treated as
stable user-facing commands.
