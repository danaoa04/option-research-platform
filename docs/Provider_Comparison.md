# Provider Comparison

Provider comparison groups normalized option records by canonical option
identifier and reports matched identities, unmatched identities, field
divergences, severity, limitations, and a deterministic checksum.

Supported fixture-backed comparisons cover ORATS, Databento, Cboe, and Polygon
through the existing reconciliation and Sprint 12C validation models. Ambiguous
identity conflicts are not silently resolved; they remain manual-review
conditions.
