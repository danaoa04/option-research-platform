## Support-bundle workflow

Before sharing diagnostics:

1. Confirm you want to generate a bundle.
2. Prefer the redacted diagnostic export path.
3. Include release manifest and build provenance metadata when available.
4. Exclude credentials, licensed payloads, and raw market data.
